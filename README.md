# CloudHomeLab
Online HomeLab Project

## Architecture

- **Oracle Cloud**: Main site with FreeIPA, HAProxy, web services
- **AWS**: Branch site with VPN gateway and web services
- **Grafana Cloud**: Centralized monitoring and logging

## Technologies

- Terraform (Infrastructure as Code)
- Ansible (Configuration Management)
- WireGuard (VPN)
- FreeIPA (Identity Management)
- HAProxy (Load Balancing)
- Prometheus + Grafana (Monitoring)
